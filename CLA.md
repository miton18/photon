## How to accept

We use the lightweight **sign-off** method (the
[Developer Certificate of Origin](https://developercertificate.org) plus this CLA).
Add a `Signed-off-by` line to every commit:

```bash
git commit -s -m "feat(ui): ..."
```

This appends:

```
Signed-off-by: Your Name <your.email@example.com>
```

By signing off, You certify the DCO **and** agree to this Contributor License
Agreement. Opening a pull request that contains signed-off commits constitutes Your
acceptance of these terms.
